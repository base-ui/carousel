
# carousel

  Base carousel ui

## Installation

  Install with [component(1)](http://component.io):

    $ component install base-ui/carousel

## Quick Start
```javascript
  var Carousel = require('carousel');
  var carousel = new Carousel(document.querySelector('.ui.carousel'));
```
